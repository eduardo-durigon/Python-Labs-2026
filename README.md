# Python-Labs-2026

Python Labs 2026

Description
This repository contains a collection of hands-on Python labs that document my learning journey as I transition into IT and cybersecurity. Each lab focuses on building practical skills through small, real-world inspired scripts.
The goal of this repository is not only to learn Python, but to apply it in a way that aligns with IT support, system administration, and cybersecurity fundamentals.

Objectives
Develop a solid foundation in Python programming
Build practical scripts for troubleshooting and automation
Strengthen problem-solving and logical thinking
Apply Python concepts to real IT and cybersecurity scenarios
Document progress consistently for learning and portfolio purposes

Structure
Each lab is organized in its own folder and typically includes:
A Python script (.py file)
A README explaining the lab
Screenshots of the code and execution
Notes and reflections

Learning Focus Areas
Python fundamentals (variables, input/output, conditionals)
Automation and scripting
Troubleshooting workflows
Cybersecurity-oriented thinking

Future Plans
This repository will continue to grow with more advanced labs, including:
Password strength checker
Log analysis scripts
Network-related tools
Basic security automation scripts

Reflection
This project represents the starting point of my transition into the IT and cybersecurity field. The focus is on consistency, practical application, and building a strong foundation over time.
