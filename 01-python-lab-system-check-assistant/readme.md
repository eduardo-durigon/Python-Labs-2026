Python Lab 01 – System Check Assistant

Description
This project is a simple Python script that simulates a basic IT support diagnostic tool.
It asks the user about system conditions such as disk space and internet connectivity, then provides a recommendation based on the answers.

What I Learned
How to use variables in Python
How to collect user input using input()
How to use conditional logic (if, elif, else)
Basic string handling using .lower()

How It Works
The script follows these steps:
Asks for a computer name
Asks if disk space is low
Asks if the internet is working
Provides a recommendation based on the responses

How to Run
Open a terminal, navigate to the project folder, and run:
python3 lab01_system_check.py

Reflection
This lab helped me understand how user input and decision-making logic can be combined to create simple automation tools.
It also gave me a better idea of how basic troubleshooting scripts work in IT environments.
